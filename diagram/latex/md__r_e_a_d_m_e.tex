In this example, {\ttfamily Abstract\+Employee} provides an abstract base class with a pure virtual function {\ttfamily ask\+For\+Promotion()}. The {\ttfamily Employee} class implements this method. This abstraction allows different types of employees to implement their own promotion logic.

These examples demonstrate the core principles of OOP in C++. Keep in mind that these are simplified examples for illustration and in a real-\/world application, these concepts are often used in more complex ways.
